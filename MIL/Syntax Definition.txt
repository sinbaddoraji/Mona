There can only be a maximum of 4 keywords/functions per line
Methods stop functionality at Error

What MIL does:

    Declear Variable
        VAR hel = "hello"
        VAR smt = SUB("Hello", "l")

    Reasign Variable value
        hel = ADD("Hello ", "World")

    Add values
        ADD(value1,value2,...)

    Subtract values
        SUB(value1,value2,...)

    Multiply values
        MUL(value1,value2,...)

    Divide Values
        DIV(value1,value2,...)
    
    Compare Values
        LESS(Value1|Value2)
        GRT(Value1|Value2)
        EQU(Value1|Value2)
        LEQU(Value1|Value2)
        GEQU(Value1|Value2)
        COMP(Value1,Value2,CompMethod)

        AND(Value1, Value2)
        OR(Value1, Value2)
        NOT(Value1)

    Input
        Input("Display string")
    
    Output
        print("Output data")

    Modifiers
        +=
        -=
        *=
        /=

    Declear Code breakpoints
        :Breakpoint name
        goto name
    
    Files
        FILE(filename)
        SAVE(filename)

    String Operations
        REP(value1, value2)
        SUBSTR(value, Start, Stop)
        LEN(value)
        SPLIT(value, splitValue) - Return Array

        REGEX
            REGEX(stringValue, Regex)
    
    Arrays
        ARR(valueName|length)
        valueName(index)
    
    Value types
        LONG -> long
        FLOAT -> float
        BYTE -> BYTE;
        INT -> int;
        DOUBLE -> double
        STR -> character
        BOOL -> boolean
        VAR -> anything
        Object
            ObjectName::value
            ObjectName
        Array
            arrayName(23)
        Null

     Cast Values
        CAST(Type|Value)
    
    Console properties
        TITLE(title)
        BG(colour)
        FG(colour)
        TOP(value)
        Left(value)

    Comments
        REM this is a comment;

    Method
        METHOD methodName
            PARM(value1, value2, value)
            Code
            Code
            Code
        ENDM
    
    Class
        CLASS className

            METHOD className
                PARM(value1, value2, value)
                Code
                Code
                Code
            ENDM

            METHOD methodName
                PARM(value1, value2, value)
                Code
                Code
                Code
            ENDM
    EndClass
